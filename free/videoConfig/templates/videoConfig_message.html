Janus configuration instructions

Server configuration

Verify that Janus is installed on the server

Verify that the streaming plugin is running and configured:
/var/snap/janus-gateway/common/etc/janus.plugin.streaming.jcfg 

      general :
      {
        admin_key = "somesecret";
        rtp_port_range = "6000-7000";
      };

Modify the admin_key to a better more secure value

FREE configuration instructions

Verify that the janus address and secret are defined in .env file:
freeweb/.env 

      JANUS_SERVER_ADDRESS=http://ip_address:port/janus
      JANUS_STREAM_ADMIN_KEY=somesecret

The JANUS_STREAM_ADMIN_KEY sould have the same value as the admin_key on janus configuration
